# Beginner Java Project: Guess The Number
"COMP1111 Fundamentals of Programming" course project. 2019

## About the project
This is a project, assigned to me from my school. It is from a fundamental java course.

Sign meanings:

- -: One of the digits of the number that you've entered is right, but in the wrong place.

- +: One of the digits of the number that you've entered is a right digit, in the right place.
